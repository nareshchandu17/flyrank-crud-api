# FlyRank CRUD API

A simple REST API built with Node.js and Express for the FlyRank Backend Internship.

## Features

- Create Task
- Get All Tasks
- Get Task by ID
- Update Task
- Delete Task
- Swagger Documentation

## Installation

```bash
git clone https://github.com/nareshchandu17/flyrank-crud-api.git
cd flyrank-crud-api
npm install
npm run dev
```

## API Endpoints

| Method | Endpoint |
|--------|----------|
| GET | / |
| GET | /health |
| GET | /tasks |
| GET | /tasks/:id |
| POST | /tasks |
| PUT | /tasks/:id |
| DELETE | /tasks/:id |
